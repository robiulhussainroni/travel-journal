# 🌏 Travel-Journal

This is a fictional travel jounal app for a fictional travel blogger to document his travel journey..

## 🌟 Overview

The goal of this project is to practice my JavaScript, OOP knowledge and keep coming back to update the project..

Project Features : 
 * User can log the travel details by location
 * User can view the log on an organized list
 * User can view the log on the map

## 🎯 Objectives

 * Practice Object Oriented Programming
 * Practice how to maintain a project over the period of time
 * Keep updating the project later down the road with new learning
